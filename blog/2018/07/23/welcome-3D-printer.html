Welcome 3D Printer!

Jul 23, 2018

In order to test the fesability of our Wind Turbine, we need to gather data about wind speed and how height affects it. In Québec, no precise information is openly available, so we deceided to build our own Weather Station and run some tests.

To build those prototypes, we bought a 3D Printer Creality CR-10S5. The Weather Station will be powered by a RaspberryPi Zero W, the Pi will gather data from multiple sensors (Altitude, Barometric Sensor, Temperature, Rain precipitation, Wind Speed and Wind Direction) and send it back to a Control Center website, where we will output and compare data into graphs.

During the week, we’ve printed a few test pieces, various tools and upgrades for the printer. Here’s the progress so far :

A Bitcoin Token to test various print quality.

The bottom case for the RaspberryPi Zero W, to secure it inside the Weather Station.

Because we will need to make a lot of solder joints, we printed this holder (the part in white) to hold the wires while we solder them.

To protect the printer, we’ve built a MDF enclosure, with In & Out fans and LED lights. The door is still missing, we’re using an curtan for the time being.

If I’m being honest, the stock fan is pretty bad, so I printed a “Fang” type of deflector to cool more effectively the printed filament.

Sometimes, we run out of filament while printing. Normaly, the printer will continue the movements, whitout really printing. To counter this, We Printed and installed a filament sensor. Now, if we run out of filament, the printer will simply pause.

Maintaining a 3D Printer requires a lot of tools, it was only fitting to print a tool rack!

To secure my personal Raspberry PI 3+, I printed a case while testing two different settings. This Pi hosts an OctoPrint server.

Because 3D printing is a very long process, usualy 8-20hrs, I use OctoPrint on my Raspberry Pi 3+ to control it remotely.
